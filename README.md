Chikka API PHP Library
=========


Chikka API PHP Library is a simple yet rich featured php library that lets you use [CHIKKA SMS API][1].


Features
----
  - Ability to send SMS/Text
  - Receive an SMS
  - Reply 
  - Fetch sms notifications
